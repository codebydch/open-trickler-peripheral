Welcome to the OpenTrickler, by Ammolytics!

From this page, you can use the following links to watch the log entries for this system as it's running, which can be useful for debugging issues, and control your trickler.

  • OpenTrickler log This monitors just the OpenTrickler code that runs on this Raspberry Pi and nothing else.
  • Screen log This monitors the OpenTrickler Screen code that runs on this Raspberry Pi and nothing else.
  • Control Panel log This monitors the OpenTrickler Control Panel code that runs on this Raspberry Pi and nothing else.
  • Raspberry Pi System log This will let you see if the system is booting up properly, and if anything has crashed, among other things.
  • OpenTrickler Servo Control Panel This will let you control your servo from a webpage using your Internet browser
  • OpenTrickler Control Panel This will let you control your trickler from a webpage using your Internet browser
